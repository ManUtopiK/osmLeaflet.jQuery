Qu'est-ce que osmLeaflet.jQuery ?

C'est un plugin jQuery (encore en alpha) qui vient vous aider à manipuler l'API de Leaflet pour afficher des cartes basées sur Open Street Map.

Capacités

  • Instances multiples
  • Choisir le niveau de zoom initial
  • Puis jouer avec votre molette ou les boutons de zoom
  • Placer des marqueurs
  • Placer des popups
  • Et donner leur contenu riche en HTML
  • Positionner une carte facilement
  • Licence BSD, faites ce que vous voulez
  • Sans contrainte de droits d'auteur
  • API simplifiée par rapport à Leaflet
  • Repose sur Leaflet et Open Street Map

Démonstration

  • Exemple
  • Code
<div id="mini_map" style="height: 140px; width: 700px;"></div>
$(document).osmLeaflet({
    target : 'mini_map',
    zoom : 9,
    latitude : 48.833,
    longitude : 2.333
}).osmLeaflet('addPopup', {
    latitude : 48.859,
    longitude : 2.333,
    text : 'Hello !'
});

Méthodes disponibles

Méthode Description
Équivaut à init
init Permet de mettre en place la carte. Appelée par défaut si pas de paramètres.

Prend les options suivantes :
  • latitude
  • longitude
  • target
  • zoom
addMarker Permet de placer un pointeur sur la carte.

Prend les options suivantes :
  • latitude
  • longitude
addPopup Permet de placer une infobulle sur la carte.

Prend les options suivantes :
  • latitude
  • longitude
  • text

Options

Nom Type Obligatoire Description
target Chaîne Oui Id de l'élément qui contiendra la carte.
zoom Nombre Non Niveau de zoom de la carte. Compris entre 1 et 18.
latitude Nombre Non Latitude de la position de la carte ou de l'élément que vous souhaitez positionner.
longitude Nombre Non Longitude de la position de la carte ou de l'élément que vous souhaitez positionner.
text Chaîne Non Contenu textuel de l'élément que vous souhaitez positionner.
