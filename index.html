Qu'est-ce que osmLeaflet.jQuery ?

C'est un plugin jQuery qui vient vous aider à manipuler l'API de Leaflet pour afficher des cartes basées sur Open Street Map.

Capacités

  • Instances multiples
  • Positionner une carte facilement
  • Choisir le niveau de zoom
  • Placer des marqueurs
  • Placer des popups avec contenu HTML
  • API simplifiée par rapport à Leaflet
  • Licence BSD, faites ce que vous voulez
  • Repose sur Leaflet et Open Street Map

Démonstration

  • Code
<div id="mini_map" style="height: 140px; width: 700px;"></div>
$(document).osmLeaflet({
    target : 'mini_map',
    zoom : 9,
    latitude : 48.833,
    longitude : 2.333
}).osmLeaflet('addPopup', {
    latitude : 48.859,
    longitude : 2.333,
    text : 'Hello !'
});
